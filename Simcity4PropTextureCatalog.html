SimCity 4 Prop and Texture Catalog

See Changelog for version details. Version: 1.0

Building upon the original the original Prop & Texture Photo Catalog started in 2011, this work includes all known dependency packs to date. This catalog was assembled by nos.17 with assistance from STEX Custodian (Tyberius06) and Cyclone Boom.

For additional information, or to reference the online version of this Catalog, visit the SimCity 4 Prop and Texture Catalog thread.

Find an issue? Have a suggestion for improvement? Drop us a message in the Catalog Thread linked above.

How to use:

  • Use the buttons below to filter the results.
  • For certain categories, an additional set of buttons provides a second level of filtering.
  • Press on each photo to open a full size version.
  • The name of each pack is linked to the location where it can be downloaded.
